# Roadmap governance

`docs/roadmap/current-status.md` is the living dashboard for `agent-run-supervisor`.

Use it to answer:

- what phase is currently supported;
- what tails remain open;
- what the next allowed request is;
- which explicit non-approvals remain in force;
- where phase evidence lives.

Do not use roadmap documents to imply implementation or live/runtime approval. A phase may start only after its scope is named and approved.
